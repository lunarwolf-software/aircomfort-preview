## Google Business Profile photos — NOT included (download not possible)

listing. They could **not** be downloaded in the build environment (the
image host was unreachable from the sandbox and browser-based export was
blocked), so per the project rules the gallery uses polished original
illustrations instead. If LunarWolf wishes to evaluate these photos, they
can be fetched manually from the URLs below — **but final use on the
website requires the owner to confirm he/she owns each photo.**

## Quarantined images — EXCLUDED, must never be used

Two photos on the Google listing belong to third parties and are excluded
from this project entirely:

1. **Competitor-watermarked photo** — an office split-unit photo watermarked
   "SUPREME AIR CONDITIONING / www.supgroup.co.za". This is a competitor's
   asset. Excluded.
2. **Unrelated billboard photo** — a roadside billboard for an unrelated
   auto-aircon business ("AIRCON", 33 Albertross Street, Rhodesfield).
   Not connected to Aircomfort. Excluded.

Recommendation: the owner should flag these via "Suggest an edit" on the
Google listing.
